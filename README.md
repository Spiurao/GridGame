## Installation & Running

### Requirements
- **Godot Engine** `v4.5`

### Running from Source
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/your-repo-name.git
   ```
2. Open Godot Engine

3. Click Import and select the project folder

4. Press ▶️ (the main scene is supposed to be Grid.tscn
